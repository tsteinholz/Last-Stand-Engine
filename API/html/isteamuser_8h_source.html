102  // Gets the latest voice data from the microphone. Compressed data is an arbitrary format, and is meant to be handed back to
103  // DecompressVoice() for playback later as a binary blob. Uncompressed data is 16-bit, signed integer, 11025Hz PCM format.
104  // Please note that the uncompressed data is not the raw feed from the microphone: data may only be available if audible
105  // levels of speech are detected, and may have passed through denoising filters, etc.
106  // This function should be called as often as possible once recording has started; once per frame at least.
107  // nBytesWritten is set to the number of bytes written to pDestBuffer.
108  // nUncompressedBytesWritten is set to the number of bytes written to pUncompressedDestBuffer.
109  // You must grab both compressed and uncompressed here at the same time, if you want both.
110  // Matching data that is not read during this call will be thrown away.
111  // GetAvailableVoice() can be used to determine how much data is actually available.
112  // If you're upgrading from an older Steamworks API, you'll want to pass in 11025 to nUncompressedVoiceDesiredSampleRate
113  virtual EVoiceResult GetVoice( bool bWantCompressed, void *pDestBuffer, uint32 cbDestBufferSize, uint32 *nBytesWritten, bool bWantUncompressed, void *pUncompressedDestBuffer, uint32 cbUncompressedDestBufferSize, uint32 *nUncompressBytesWritten, uint32 nUncompressedVoiceDesiredSampleRate ) = 0;
